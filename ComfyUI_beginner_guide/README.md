# ComfyUI 初心者ガイド

ComfyUI を初めて触る方向けの操作手順書です。**画像生成** と **音楽生成** の両方を、自分の PC で動かせるようになることを目指します。

> **目次は [00_index.md](00_index.md) からどうぞ。** 各章はそこから順番にリンクされています。

## ディレクトリ構成

```
ComfyUI_beginner_guide/
├── README.md                  ← このファイル（概要のみ）
├── 00_index.md                ← 目次 + 環境情報 + 用語ガイド
├── 01_introduction.md         ← 第1章 はじめに
├── 02_ui_tour.md              ← 第2章 画面ツアー
├── 03_setup.md                ← 第3章 セットアップ
├── 04_image_generation.md     ← 第4章 はじめての画像生成 (SDXL)
├── 05_prompt_tips.md          ← 第5章 プロンプトを変えて遊ぶ
├── 06_audio_setup.md          ← 第6章 音楽生成の準備
├── 07_stable_audio.md         ← 第7章 Stable Audio
├── 08_ace_step.md             ← 第8章 ACE-Step
├── 09_troubleshooting.md      ← 第9章 困ったときに
├── 10_glossary.md             ← 付録 用語集
└── screenshots/               ← スクリーンショットと生成サンプル
```
