# ComfyUI 初心者ガイド — 目次

ブラウザで `http://localhost:8188/` を開いた状態から、自分のPCで **画像** と **音楽** を生成できるようになるまでを案内する手順書です。

## 想定する読者

- ComfyUI を初めて触る
- 「ノード？ ワイヤー？」と聞いて少しひるんでしまう
- とりあえず画面を見ながら「ここをクリックすればよい」を順番に教えてほしい

## 動作確認した環境

| 項目 | 値 |
|---|---|
| ComfyUI | v0.3.60 |
| Frontend | v1.26.13 |
| GPU | AMD Radeon VII (VRAM 16GB) / ROCm 5.7 |
| Python | 3.10.12 |
| PyTorch | 2.3.1+rocm5.7 |
| 実行形態 | Docker (image: `comfyui-rocm57:latest`) |
| アクセスURL | http://localhost:8188/ |

NVIDIA GPU や別バージョンの ComfyUI でも、ほとんどの手順はそのまま使えます。

## 章をまたいで使うキーワード

- **ノード (Node)** — 「モデルを読む」「テキストを変換する」などの **1つの仕事を担当する箱**
- **ワイヤー (線)** — ノード同士をつなぐ線。前のノードの出力を次のノードの入力に流す
- **ワークフロー** — ノードとワイヤーで組まれた **一連の処理の図**

詳しい説明は [付録：用語集](10_glossary.md) にあります。
